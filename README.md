# ProjectRPG

##RPG Prototype with AI Fight System developed with Unreal Engine 4

AI FIGHT SYSTEM:
- AI Brawl System: enemies tries to surround the player smartly, without boumping into each other
- Different hit reaction based on bodyparts hit
- AI and life based on enemy level
- Targeting System
- Hearing and Sight Sense
- Investigation pattern
- Patrol Paths
- Advanced Behavior Tree with custom tasks, services and decorators
- Dodge, strafe, evade, block

Movement control:
- Root motion animation to all Mixamo animation
- State machine with blend spaces
- Control movement and rotation based on velocity
- Jump system: 
   * still and in motiong
   * camera shake on landing
   * different gravity function curve
- Crouch system with heigth based on velocity
- Sprint with procedual leaning, camera shake on step and camera arm movement
- Different vault animation based on velocity with capsule collision handling
- Hang/Climb system:
    * different hang animation based on wall height
    * hanged side movement
    * stop on edges
    * side jump
    * wall run
